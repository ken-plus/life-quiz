# Life Quiz

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ken-plus/life-quiz)

## 3 步驟上線

1. 點擊上方綠色按鈕 "Use this template"
2. 點擊藍色 "Deploy with Vercel" 按鈕
3. 完成！

## 如何修改內容

1. 進入你的 GitHub repo
2. 找到要修改的檔案
3. 點擊鉛筆圖示編輯
4. 改完後點 "Commit changes"
5. 等待 1-2 分鐘網站自動更新
